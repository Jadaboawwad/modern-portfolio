## AI Work thumbnails (Perchance prompts)

### How to use
- Generate each image in Perchance (Text-to-Image).
- Use **resolution 768x512 (landscape 3:2)** if available.
- **No text**, **no logos**, **no watermarks**.
- Save into `modern-portfolio/public/work/` using the filename listed below.

### Shared style (append to each prompt)
Dark purple/navy tech aesthetic with red/orange accent glow, clean modern composition, high contrast, subtle 3D glow, abstract UI, digital art.
